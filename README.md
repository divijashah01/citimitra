CitiMitra is a government-aligned digital platform that converts citizen reports and willingness to help into **verified, skill-based civic volunteer tasks** using AI, geofencing, and automated validation.  
It bridges the gap between **citizens and local governments** through transparent dashboards and structured civic engagement.

---

**Key Features**
- Aadhaar-linked citizen verification  
- AI-powered skill profiling & task matching  
- Photo-based issue reporting  
- Geofenced task check-ins for authenticity  
- Automated reward & badge system  
- Government analytics dashboard & heatmaps  

---

**Tech Stack**

**1. Frontend**
 - Flutter 4.0 (Dart)
 - BLoC State Management

**2. Backend & Cloud**
 - AWS Lambda (Python)
 - Amazon API Gateway
 - AWS Cognito (Authentication)
 - Amazon DynamoDB (Database)
 - Amazon S3 (Image Storage)

**3. AI & Verification**
 - Amazon Bedrock (Claude / Nova Pro)

**4. Services & Tools**
 - AWS Location Services (Geofencing & Maps)
 - Amazon SNS + Firebase Cloud Messaging (Notifications)
 - Sandbox.co.in API (Aadhaar Verification)
 - VS Code, Figma, GitHub
